Para compilar abra um terminal, navegue até a pasta Sierpinski e execute o arquivo "Makefile" com o comando "make".

Para executar abra um terminal, navegue até a pasta Siepinski e execute o arquivo "Sierpinski" com o comando "./Sierpinski" ou "./Sierpinski <tamanho>",
substituindo o "<tamanho>" pelo tamanho da aresta do maior triangulo.
